## Description
A platform where users trade skills instead of money. For example, I teach you 30 minutes of Python, and you teach me 30 minutes of Guitar.
## Features
1. Account:
   1. Register new account
   2. Forget password
   3. Login
   4. Update user information: Personal info
   5. Logout
2. Skill set:
   1. Add/Remove skill
   2. View list of skill sets
3. Skill swap request:
   1. Request data: Owner, Offering skill, Skill want to learn, Status, Selected offers, list of offers.
   2. Place a request: Define known skill and which skill to learn
   3. Search a request: Filter by skill set / Skill category
   4. Offer/Accept/Close/Reject a request
   5. List of requests: Display requests in status "Requested" (for request owner), "Offering" (for offerers), "In progress" (for both stackholders)
4. (Optional) Real-time notification system when someone requests a swap.
## Screen
1. Homepage:
2. Account:
   1. Register new account
   2. Login
   3. User information
3. Skill management
   - Display my skill sets.
   - Add/Remove/Edit skill sets.
4. Swap request management:
   1. List of my request:
      - Display swap requests that in status "Requested" and I am the owner.
      - User can place a new swap requests.
   2. Search for swap request: Search by skill set / Skill category
   3. List of ongoing request: Display request that in status "In progress"
   4. Swap request detail:
      - Display request information, list of offering (display for owner)
      - Offerer: Offer a request if it is a match
      - Owner: Accept/Delete an offer. Close a request.
      - If the request is in status "In progress", both user can reject the request.
